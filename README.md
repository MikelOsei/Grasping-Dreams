# Grasping Dreams
Grasping Dreams utilizes pre-processed EEG data as input, with an interactive component that invites viewers to clear away the dream space, 
revealing the underlying activity of the human brain. 

Please see my artist statement for more details.
Created as Assignment 2 for UWaterloo's CS/FINE 383 Winter 2026 offering.

## Controls: 
* Press H to show/hide the settings GUI
* Press SPACE to toggle between scans
* Use your hands and camera to interact with the artwork.
